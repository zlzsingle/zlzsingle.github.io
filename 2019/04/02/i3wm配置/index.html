i3wm个人配置

安装i3wm

  • ubuntu 系统

安装配置应用软件

  • 截图软件,使用 flameshot ubuntu18.04以上系统可安装

  • 锁屏软件 i3lock

  • 图片浏览器 feh

配置

  • 设置功能键,一般配置的是windows键或super键, alt键在配置中写作mod1,回车键Enter写作Return

  • 打开终端 $mod + 回车

  • 打开桌面菜单 $mod + d

  • 重新加载i3配置 $mod + Shift + c

  • 重启i3 $mod + Shift + r

  • 退出i3 $mod + Shift + e

  • 截图 ctrl + $mod + a

  • 锁屏 $mod + l

  • 开启一个工作区 $mod + (1 ~ 0)

  • 移动到某个工作区 $mod + Shift + (1 ~ 0)

  • 状态栏配置

  • 启动执行sh脚本

  • 桌面定时切换壁纸
